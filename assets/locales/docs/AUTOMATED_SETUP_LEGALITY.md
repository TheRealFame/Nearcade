# Setup Legality

NearsecTogether includes powerful features for running headless, unattended game servers ("Auto-Hosts").

### Acceptable Use
You are fully responsible for the games and content you host via Nearsec. 
- You must own the rights or licenses to the software you are hosting.
- The Nearsec Auto-Host feature simply forwards inputs to a local application.

### Commercial Use
If you intend to use NearsecTogether for commercial arcade setups or paid cloud gaming services, you must comply with the EULAs of the games you are streaming. Nearsec is just the streaming protocol; it does not grant you commercial rights to third-party games.

### Liability
The authors of NearsecTogether are not liable for any account bans, TOS violations, or legal disputes resulting from the unattended hosting of video games.
